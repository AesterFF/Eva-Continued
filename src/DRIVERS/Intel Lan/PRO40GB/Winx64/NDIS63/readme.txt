The driver package files in this folder can be used to install drivers for Intel(R) Ethernet 700 Series adapters on the following operating systems:
   *  Microsoft Windows Server* 2012 (x64 Edition)
   
The driver package supports devices based on the following controllers:
  * Intel(R) Ethernet Controller X710
  * Intel(R) Ethernet Controller XL710
  * Intel(R) Ethernet Controller XXV710
